# ⚖️ FM Legal Studio  
A web application built with **Django**, designed for a law firm with modern functionality, clean structure, and an intuitive interface for presenting legal services.

## ✨ Features

**Dynamic pages** generated using *Django Templates*  
**Contact form** that stores client messages in the **database**  
**Admin panel** for managing inquiries and client messages  
Clean and organized **backend + frontend** structure  
Proper handling of static files (**CSS**, **JavaScript**, **images**)  
Simple and extendable URL routing  


## ⚙️ Technologies Used

**Django 4.x**  
**Python 3.x**  
**HTML / CSS / JavaScript**  
**SQLite** *(default development database)*
